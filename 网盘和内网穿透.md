## 内网穿透知名开源方案
1. [nps 内网穿透代理服务器](https://github.com/ehang-io/nps)  一款轻量级、高性能、功能强大的内网穿透代理服务器。支持tcp、udp、socks5、http等几乎所有流量转发，可用来访问内网网站、本地支付接口调试、ssh访问、远程桌面，内网dns解析、内网socks5代理等等……，并带有功能强大的web管理端
2. [Ngrok](https://ngrok.com/docs/getting-started/) 一个非常流行的开源内网穿透服务，支持多个操作系统和多种协议，具有简单易用的界面和丰富的功能。[源码](https://github.com/inconshreveable/ngrok.git)

使用ngrok的内置文件服务器为本地目录提供服务

ngrok可以使用自己的内置文件服务器为本地文件系统目录提供服务，不需要单独的服务器。在指定转发URL时，您可以使用file://方案提供文件。

通过身份验证在计算机上共享一个文件夹
```
ngrok http --basic-auth="user:password" file:///Users/alan/share
```
